Contao Music Bundle
=======================

[![](https://img.shields.io/packagist/v/numero2/contao-music.svg?style=flat-square)](https://packagist.org/packages/numero2/contao-music) [![License: LGPL v3](https://img.shields.io/badge/License-LGPL%20v3-blue.svg?style=flat-square)](http://www.gnu.org/licenses/lgpl-3.0)

About
--

Add Spotify or Apple Music player to your website.

System requirements
--

* [Contao 5.0](https://github.com/contao/contao) (or newer)

Installation
--

* Install via Contao Manager or Composer (`composer require numero2/contao-music`)
* Run a database update via the Contao-Installtool or using the [contao:migrate](https://docs.contao.org/dev/reference/commands/) command.
